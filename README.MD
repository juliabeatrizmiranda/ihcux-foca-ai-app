Conceito de Design e Fluxo de Usuário: foi criado com uma interface simples e minimalista para reduzir distrações e ansiedade. O usuário entra no aplicativo, escolhe um som ambiente e o tempo de foco, configura os aplicativos que serão bloqueados e inicia uma sessão de foco em grupo. Durante a atividade, acompanha o cronômetro e o crescimento da árvore como incentivo positivo.
Escolha de Componentes
° Botões rápidos para seleção do tempo de foco.
° Cards com ícones para escolha dos ambientes.
° Switches on/off para bloqueio de aplicativos.
° Barras de progresso para mostrar ranking e evolução do grupo.
